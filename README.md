# Coding Challenge


#### Setup

Data stored in YAML file under data/employee.yaml file

Before start application, pls do 

```
composer install
```

#### Start application

```
./bin/console app:calculate:vacationdays <year>
```

For example :

```
./bin/console app:calculate:vacationdays 2017
```

#### Tests

To run unit tests 

```
./vendor/bin/phpunit
```
